# mumbela-digital-labs.github.io

## Getting Started

### 1. Prerequisites
- Ruby with DevKit installed and on PATH.

### 2. Clone the repository from GitHub:

- Clone using GitHub CLI:
```bash
$ gh repo clone mumbela-digital-labs/mumbela-digital-labs.github.io
```

- Or via https:
[https://github.com/mumbela-digital-labs/mumbela-digital-labs.github.io.git](https://github.com/mumbela-digital-labs/mumbela-digital-labs.github.io.git)


### 2. Install the jekyll gem:

```bash
$ gem install bundler jekyll
```
### 3. Serve the static site:
```bash
$ bundle exec jekyll serve
```
